Project Overview

SolveMyIndia is a non-profit initiative designed to bridge the gap between citizen-led ideas and policy-making. The platform provides a democratic space for young people to propose, discuss, and vote on solutions to pressing issues in India. My primary responsibility was to design and build the entire MERN stack application, focusing on real-time capabilities and a highly accessible user interface.

Key Features

  • Democratic Proposal System: Registered users can submit detailed proposals, which are then open for public voting and debate.
  • Real-time Discussion Forums: Each proposal has a dedicated discussion thread powered by Socket.io, allowing for live chat and community engagement.
  • User Authentication & Profiles: A secure user management system allows individuals to track their proposals, votes, and contributions.
  • Impact & Analytics Dashboard: A feature for moderators to track proposal popularity, user engagement metrics, and generate reports on trending topics.

Technical Challenges

The main technical challenge was implementing a robust real-time voting and chat system at scale. Using Socket.io, I created a system of rooms for each proposal, ensuring that discussions were isolated and efficient. Another focus was on data integrity, preventing duplicate votes and ensuring the democratic process was fair and transparent. The UI was built with accessibility in mind (WCAG standards) to ensure the platform could be used by people of all abilities.
