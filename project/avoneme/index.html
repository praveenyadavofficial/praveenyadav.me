Avoneme Creative Studio

A digital-first brand identity and dynamic web platform for a modern creative agency.

Role: Creative Director & Lead Developer
Date: 2025 — Present
Tech Stack: React, Node.js, Figma, Framer Motion
Status: In Progress
Visit Live Site

Project Overview

Avoneme is a startup creative agency requiring a powerful online presence to showcase their portfolio and acquire clients. My role encompassed the entire project lifecycle, from brand strategy and UI/UX design in Figma to full-stack development. The core challenge was to create a platform that was not just a static portfolio but an interactive experience reflecting the agency's creativity and technical skill.

Key Features

  • Dynamic Service Showcase: An interactive section detailing the agency's offerings, using animations to engage users.
  • Filterable Portfolio Gallery: A performant React-based gallery with smooth transitions and lazy-loaded images to handle a growing body of work.
  • Client Inquiry System: A multi-step form connected to a Node.js backend that captures detailed client requirements and automates initial communication.

Design and Development

The process began with wireframing in Figma to map user flows for potential clients. We chose a minimalist aesthetic to ensure the creative work was the star. For development, a React frontend provided the required interactivity, while an Express.js server handles API requests and form submissions, creating a robust and scalable foundation.
